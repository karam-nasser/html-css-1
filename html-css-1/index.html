Gematria, Numerology Code

Gematria is a numerological system by which letters correspond to numbers. The system, developed by practitioners of Kabbalah, derived from Greek influence.

How It Works

In gematria, each letter is represented by a number (for example, A = 1, B = 2, etc.). One can calculate the numerical value of a word by adding together the values of each letter in it.
